# Data Analysis and Visualization Methods (DAVM)

A comprehensive repository for Data Analysis and Visualization Methods (DAVM) coursework and projects. This repository contains datasets, PowerBI workbooks, Tableau visualizations, and case studies focused on deriving insights from data.

## Contents

- **Dataset**: Raw data used for analysis and visualization.
- **PowerBI**: Interactive dashboards and reports created using Microsoft Power BI.
- **Workbooks**: Excel and other data workbooks for pre-processing and exploratory data analysis.
- **Group Project & Case Studies**: Collaborative analysis projects covering various business domains.
- **Previous Year Solutions**: Reference materials for course assessments and problem-solving.

## Key Visualizations

- **Customer Behavior Analysis**: Deep dives into user demographics and purchase patterns.
- **Business Performance Dashboards**: Metrics for growth, retention, and operational efficiency.

## Tools Used

- Microsoft Power BI
- Tableau
- Microsoft Excel
- Python (for data cleaning)

## How to Use

Navigate to the respective folders to explore the `.pbix` (Power BI) or `.twb/.twbx` (Tableau) files. Datasets are provided in CSV/Excel format within the `Dataset` folder.
